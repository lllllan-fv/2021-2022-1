# 用户相关

- 用户：登录系统的一个账号，是购买车票的主体。

- 乘客：真实存在的一个个体，是乘坐火车的主体。

# 火车相关

- 火车信息表-火车：实指一列火车实体。
- 火车往返时刻表：规定该列火车每天应该在什么时间从哪里发往哪里。
- 火车运行时刻表：指该列火车在某天运行的具体行为。
